Week 07 Outcomes

  • Create single table queries
  • Code statements, commands, clauses and keywords that are used in single table queries
  • Code using arithmetic, comparison and logical operators

Before the Week Begins

  • Students will go over preparation materials and complete the preparation check before the first class of the week.
  • Consider reviewing the teaching tips and suggestions in the CIT111 - W07 Course Group discussion https://byui.instructure.com/courses/70298/discussion_topics/1457734.

During the Week

First of the week:

  • The students should have completed the preparation materials and check before the first class of the week and be ready for learning, activities, and discussions.

    (W07 - Query Art DB)

  • Because not all student inserted the data into the database properly, have them import the v_art database from the dump file so they all have the same database to query from.
  • They will also use a new bike shop database this week. Have the also import that database as well. Reverse engineer this new bike database so the students can see the ERD and how it is designed with Database - Reverse Engineer.
  • With the introduction of the SELECT statement coding order (SELECT, FROM, WHERE, ORDER BY, LIMIT) it's good to go over the order of execution order (FROM, WHERE, SELECT, ORDER BY LIMIT). This helps explain why you can use an alias in ORDER BY but not WHERE.
